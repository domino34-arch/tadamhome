Comment ça marche

Trois gestes pour composer
un intérieur augmenté.

01

Scannez

Pointez un QR code sur une fiche produit, en showroom ou en ligne. Le modèle 3D se charge instantanément, sans application à installer.

02

Configurez

Changez le matériau, la couleur, les dimensions. Visualisez la finition exacte que vous voulez avant même de toucher le produit physique.

03

Visualisez en AR

Posez le meuble dans votre vrai salon, à l'échelle réelle. Composez plusieurs pièces de plusieurs marques dans la même scène, partagez-la, archivez-la.
